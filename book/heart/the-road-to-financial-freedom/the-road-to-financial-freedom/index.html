财务自由之路

说明：以下都是 prompt 记录了一些正面语句，负面案例并未被我记录，感兴趣的可自行观看原书。读这本书的最大感受是把我的思想形式化了，我做不到说的这么精准和详细，里面没有让我觉得突兀的观点。

核心感悟

投资并不是赚钱，是做人。我为什么没有从小研究巴菲特段永平的思考方式，真是后悔莫及，希望现在没有晚。

我知道了竞赛对工作的误导：竞赛重要的是思考，而工作重要的是学习。说的直白点就是直接学，越快学，学的越多越好。普通人遇到的问题全部有标准答案，直接照做即可，为什么一个普通人需要觉得自己比巴菲特更智慧呢？

我认识很多人，没看过这本书，却践行了这本书的道理。

自我分析

自我分析处于什么样的财务状况。

自信与乐观

  • 乐观主义让你看到事务的积极面
  • 自信给你一种战胜事物黑暗面的信心
  • 自信可以让一个人在自身经历的基础上，知道遇事可以靠自己
  • 你的财务状况应该成为一个你不惧怕任何阻碍的证明
  • 一个没有自信的人，只是一个一事无成一无所有的无名小卒

金钱观

  • 管理你的财务状况，使它服务于你，而非给你带来麻烦
  • 金钱可以使你生活艰难，也可以使你生活轻松
  • 金钱是善物

你的思维方式铸就了你今天的样子，这样的思维方式却并不能让你变成你想成为的样子。

生活的五个领域

生活 = 健康 + 财务 + 关系 + 情感 + 人生意义

都同样重要。如果别人用锤子击打你的一根手指，你会觉得”没事 我还有4根”还是觉得”这根手指疼死了”？

工作与赚钱

  • 想短期富有的人本质需要的是赚钱机器，而不是穷尽一生当一台赚钱机器
  • 不做感兴趣事情的原因是缺钱
  • 找一份既感兴趣又赚钱的工作
  • 整天工作的人是没时间赚钱的
  • 优化自己还是得过且过，要把5天工作日当成成就感，而不是对周末的打扰
  • 要么别人规划自己的时间，要么自己规划自己的时间

财富的四大支柱

将梦想、目标、价值观、策略联系在一起，4支柱是行为准则，财富在上面建立。

梦想

如果你拥有无限的时间和金钱，你会做什么？梦想就是快乐。

价值观

对你来说真正重要的东西是什么？为什么这些东西很重要？

  • 需要和梦想、目标一致，减少矛盾
  • 价值观是一种可以改变的选择
  • 你拥有改变的自由
  • 只有当你清晰地确定指导自己的价值观后，才能掌控自己的生活

目标

  • 你想成为什么样的人？
  • 你想要做什么？
  • 你想要拥有什么？

梦想确定目标。

策略

你具备能协助你获得自己想要的东西的知识、能力、计划吗？

  • 应对债务
  • 知识获得金钱
  • 如何快速提高收入
  • 留住钱才能富有
  • 高收入不会富有
  • 钱生钱
  • 规划财务目标
  • 确保目标执行

需要一个引导你致富的人。创造一个逼迫你一定坚持的环境，为了达成目标。

成功与幸福

  • 成功表示你拥有你想要的东西
  • 幸福表示你热爱你拥有的东西

富有并非难事，等到一个人变富有，就会奇怪为什么没有更多人富有。

一切都要从我们掌控生活的态度开始。

责任

  • 有责任就有权力，责任就是反应
  • 需要对事情的判断和反应负责
  • 无论我们遇到什么，我们都可以决定怎么应对
  • responsibility = response + ability，有技巧地进行回应
  • 以争吵应对争执，以冲突应对挑衅，绝非最佳的问题解决方式
  • 自己掌控自己的生活
  • 抬起木棍的一端，它的另一端也随之而起
  • 自己掌控未来，还是让自己的错误掌握未来

梦想成真的自信来源于你的过往。

拓宽可控领域

如果总是觉得问题由外界因素引起，所以它不可控，这种想法本身就是问题。

四个方法

  1. 走出舒适区

    • 挑战上瘾，类似于旅游，爱上新环境
    • 船在港口有安全感但这不是造船的目的
    • 用危机来保持活力
    • 舒适区就是可控领域

  2. 问题

    • 问题本身就是机会
    • 逃避问题不会变得富有
    • 问题处于个人范畴内却不处于可控领域
    • 需要问题清单

  3. 正确的提问方式

    • 提问质量决定生活质量
    • 不要问能不能，要问怎么做
    • 关注能做的事可以带来收益
    • 我们现在能做什么？

  4. 决策顺序

    • 我为什么要这样做？决定是否要做
    • 我应该怎么做，出现下一个困难如何解决
    • 一直牢记不放弃的理由

  5. 扩大小世界

    • 小世界包括影响你的、对你重要的、你感兴趣的事物

潜能与成功

我们充分利用自己的潜能时，会获得极大的满足感。我所理解的成功是成为最好的自己。全力以赴会感受到前所未有的活力。

金钱远比大多数人想象的美好，财富积累也远比大多数人想象的要简单。

贫穷是自行产生的，在你什么都不处理的时候。

责任意味着：没有任何东西能改变你的态度或品格，因为你是按照自己本性对事情做出反应。你可以决定自己以哪种方式生活，这种态度能让自己幸福，成为最好的自己。如果你勇于承担责任，那么你就能在7年内变得富有。

发生变化的5个层次

  1. 对现状不满
  2. 期待的结果没有出现
  3. 技巧产生了作用
  4. 打交道时反思世界观
  5. 改变对自身的认知可以带来巨大的变化

在失败中看到积极的一面，这并不容易。

干等着奇迹发生是一件愚蠢的事情。

持续学习与成长

  • 持续不断地学习和成长是人生信条
  • 成长就是生命
  • 一个人是他读过书的总和
  • 结实有趣的人，互相推荐书

笔记本系统

  • 成功日记本：失败的记忆更持久所以需要记录成功
  • 思想笔记本：创意
  • 关系笔记本：快乐
  • 知识笔记本：教训

将自己看做一个重要的人，开始记录关于自己的故事。

信心与冒险

  • 信心不足导致不承担风险，导致一事无成
  • 没有冒险，没有成长
  • 环境不有助于信心培养，所以成功笔记会让你意识到有多么优秀
  • 自信决定期望，期望决定成功

学习方式

  • 讲座是plus版读书，多模态地快速经历许多东西，开放式地多角度思考事物
  • 我们也教育付出的代价，相比于无知愚昧付出的代价不值一提
  • 美国有讲座市场，这个待考察是什么东西

  • 没有必胜，只有机会

  • 奇迹发生在变化

运气

  • 金钱上的好运 = 储蓄 + 识别机会 + 抓住机会
  • 好运都是以机会的形式敲门的
  • 必须马上采取行动
  • 承担责任的人，行动迅速

贫穷的原因

  • 贫穷也是因为坚持一些原则而产生的
  • 没钱的人对财富没有明确定义
  • 需要想象自己的成功人士，有这样的画面

目标设定

  • 大目标更容易达到，目标越大，问题越小
  • 为自己设立有生之年无法实现的目标
  • 积累了大量财富的人，并非遵守纪律的典范，没有其他选择，因为无法穷困潦倒或庸庸碌碌
  • 你有能力使你的目标成为绝对必需品
  • 不实现这些目标你将无法幸福，一成不变的生活对你来说是巨大的痛苦。你一定要实现自己的目标。

丘吉尔：绝不、绝不、绝不、绝不放弃

如果我们设立极限，我们就会达到极限。

成功可以重复

  • 永远不要让替你承担责任的情况出现
  • 你可以随时重复你的成功，你可以创造出成功所需的客观条件
  • 你既然成功过一次，就可以再次取得成功
  • 你如果认为成功是不可重复的，就失去了提升自我价值的机会
  • 因此，你取得成功之后，快速地再次取得成功也是相当重要的

没钱的人不准备付出110%的努力。

常见误区

  • 爱找借口的人总会找到借口
  • 误区：将来我也会对现在拥有的一切感到满足
  • 误区：假如我能全力以赴，我就会成为最优秀的
  • 无论是谁，将最后一个可能的借口都摒弃掉，必将取得成功
